HTML要素について

<p>について

<p>要素はパラグラフ(paragrah)を意味する。
適切に使用し人が読みやすい文章を書くように心がけよう!

<ul>と<li>について

<ul>と<li>はlistを作る。この要素を使えば簡単にリストを作れるぞ！
<ul>のcontentに<li>で囲まれたモノがリストの要素になる

  • りんご
  • みかん
  • ばなな
<ul>の代わりに<ol>を使用するとリストに順序がつく！
  1. りんご
  2. みかん
  3. ばなな
またリストを入れ子にすることもできるのだ！
必要に応じてリストを作成しよう！
  1. りんご
    • 世界一
    • アリエーヌ
    • ぜスター
  2. みかん
  3. ばなな

<em>について

<em>は文字をイタリックにして強調できる。
さらにこの要素を適用した文字は読み上げソフトを使用した時の音声も変化する。

  • This is a real pen.
  • This is a real pen.
このように二つの文を比較すると下の文のrealが強調されていることがわかる。

<strong>について

<strong>についてはこの要素で囲まれた部分をbold(太文字)にできる。

<dl>と<dt>と<dd>について説明する
説明リストと呼ばれる。用語の定義やQAなどに適している。

question : りんごてなんですか？
answer: りんごは果物の一種で赤い。

<addr>について

<addr>は略語の解説をしたり簡単な注釈を入れたりするのに便利である。
addr要素で囲まれた文字にマウスオーバーするとaddrのtitleに設定した値が表示される。
例えばマクドて言うよね？
